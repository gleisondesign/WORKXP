Perguntas frequentes

Perguntas frequentes

O que é o WorkXP?

WorkXP é uma ferramenta gratuita de produtividade gamificada que combina tarefas, fases, Pomodoro, foco, timeboxing, XP, níveis, conquistas e importação de projetos por IA.

O WorkXP exige cadastro?

Não é necessário criar uma conta para começar. O projeto é salvo localmente no navegador por meio de localStorage.

Posso levar meu projeto para outro computador?

Sim. Use a função Exportar para baixar um arquivo JSON e depois use Importar no outro navegador ou computador.

O WorkXP funciona em inglês?

Sim. A interface possui versões em português brasileiro e inglês, com detecção do idioma do navegador e alternador manual.

Para quem a ferramenta é indicada?

Ela pode ser usada por estudantes, freelancers, criadores, desenvolvedores, designers, profissionais autônomos, equipes pequenas e pessoas com projetos pessoais.
